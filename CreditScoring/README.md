# Кредитный скоринг. Бинарная классификация.

## Задача

Помочь банку принять решение о выдаче займа клиентам

## Данные

Данные представляют собой более миллиона записей с более чем 100 признаками и одной целевой переменной: «итоговый*статус*займа». Значения: 0 – выплачен, 1 – не выплачен

## Метрика

Метрика соревнования – roc-auc
